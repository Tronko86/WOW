# The Rookery — Panoramica Mythic+

Dungeon corveria corrotto da void magic a Dornogal, Isle of Dorn. 3 boss con meccaniche Storm/Void. Progressione verticale multi-piano con discese e corrupted Stormriders.

## Caratteristiche Generali

**Durata**: ~25-30 minuti  
**Boss**: 3 (Kyrioss, Stormguard Gorren, Voidstone Monstrosity)  
**Tipo dungeon**: Vertical multi-floor + void corruption theme  
**Dispel richiesto**: Sì (Chaotic Corruption, Unstable Charge)  
**Meccanica unica**: Interactive Stormrook buff, multi-floor navigation, Void Shell break mechanics

## Struttura del Dungeon

### Part 1: Storm's Roost (Floor 1)
- **Boss 1**: Kyrioss (Corrupted stormrook con Lightning Torrent)
- **Post-boss**: Discesa con feathers Stormrook → libera Captured Stormrooks per buff

### Part 2: Stormrider Barracks → Crashing Storm Hall (Floor 3-4)
- **Boss 2**: Stormguard Gorren (Corrupted earthen con void powers)
- Trash waves corrupted Stormriders

### Part 3: Abandoned Mines (Floor 5)
- **Boss 3**: Voidstone Monstrosity (Amalgamation void boss, shield break mechanics)

## Stile di Combattimento

- **Movement**: alto (Lightning Torrent rotations, Dark Gravity pulls, Null Upheaval swirlies)
- **Coordination**: richiesta (Chaotic Corruption spread, Unstable Charge discharge, Voidstone Fragments management)
- **Nature/Shadow damage**: predominante
- **Add management**: Voidstone Awakened critical threat se non gestiti

## Cosa Causa Più Wipe

1. **Lightning Torrent** (Kyrioss) → non evitare rotating beams = stun + wipe
2. **Chaotic Corruption** (Stormguard Gorren) → chain jump senza spacing = raid-wide explosion devastante
3. **Voidstone Awakened** (Voidstone Monstrosity) → Corruption Overload non killato = stacking damage exponential wipe
4. **Lingering Void pools** (Stormguard Gorren) → durano 2 min, limitano spazio fight drasticamente
5. **Unstable Charge** (Kyrioss) → discharge fuori Stormheart tempest = raid damage spike

## Consigli Generali per Ruolo

### Tank
- **Kyrioss**: posiziona boss per ottimizzare movement durante Lightning Torrent phase
- **Stormguard Gorren**: gestisci Dark Gravity positioning, evita Lingering Void pools
- **Voidstone Monstrosity**: mantieni aggro durante Void Shell break, avoid Oblivion Wave

### Healer
- **Attenzione a dispel timing**: Unstable Charge (Kyrioss), Chaotic Corruption (Gorren)
- **Lightning Torrent**: raid-wide AoE healing spike ogni 3 sec per 15 sec (M+)
- **Voidstone Awakened**: Corruption Overload stacking = healing check crescente
- **Dark Gravity**: moderate sustained damage + knock-back spike healing

### DPS
- **Priority**: Voidstone Fragments PRIMA che diventino Awakened (15 sec window)
- **Stormrider's Charge**: usa buff movement per detonare Fragments
- **Void Shell**: break shield veloce per Storm's Vengeance stun (20 sec window)
- **Unstable Charge**: discharge in Stormheart central tempest (Kyrioss)
