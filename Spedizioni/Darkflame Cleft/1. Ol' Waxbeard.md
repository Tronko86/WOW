# Ol' Waxbeard — Boss 1

Leader kobold che evoca ondate di lavoratori e minecart. Meccanica chiave: Luring Candleflame (fixate add) e Reckless Charge con Cave-In.

## Panoramica boss

**Tipo**: Add Management + Positioning Boss  
**Meccanica chiave**: Luring Candleflame (fixate add su target), Reckless Charge (Wick charge + Cave-In), minecart dinamiche  
**Difficoltà principale**: gestire add waves + evitare Cave-In rocks + minecart timing (M+)

## Meccaniche chiave

1. **Luring Candleflame** (Richiamo Candela)
   - Ol' Waxbeard **marca un giocatore**
   - **TUTTI gli add** (Menial Laborers) **fixano il target marcato**
   - Add ignorano threat e vanno sul target marcato
   - Target marcato deve **kite add** in posizione corretta
   - Strategia: stack add per AoE cleave

2. **Reckless Charge** (Carica Spericolata)
   - **Wick** (add assistente di Waxbeard) **charge verso il giocatore più lontano**
   - Charge causa **Cave-In** lungo il percorso
   - **Cave-In**: rocce cadono dal soffitto
     - Alto Physical damage a chi colpite
     - **ISTAKILLA add kobold** se colpiti
   - Usa Cave-In per killare add Menial Laborers

3. **Rock Buster** (Frantumasassi)
   - Colpo pesante tank melee
   - Infligge alto Physical damage
   - Lascia debuff: **aumenta Physical damage taken** stackabile
   - Tank swap se stack troppo alti (M+ high keys)

4. **Menial Laborers** (Lavoratori Umili)
   - Add evocati in waves
   - Ottengono buff **Crude Weapons** dopo spawn
   - **Crude Weapons**: stacking Physical DoT su target
     - Più add vivi = più stack = più damage
   - DEVONO essere killati velocemente

5. **Underhanded Track-tics** (Minecart Dinamica) [Mythic+ only]
   - Ol' Waxbeard evoca **minecart con dinamite**
   - Minecart ha **20 secondi** timer
   - Se **NON distrutto** entro 20 sec → **esplosione raid-wide** devastante
   - Priority interrupt: KILLA minecart entro 20 sec

6. **Minecart/Reckless Charge interaction**
   - **Cave-In rocks** istakillano add
   - **Wick charge** istakilla add lungo percorso
   - **Minecart** istakillano add se colpiti
   - Usa meccaniche per cleave add efficacemente

## Cosa interrompere / dispellare / evitare

- **Interrupt**: non necessario (nessuna cast critica)
- **Killa priority**: 
  - **Minecart dinamiche** (M+) → max 20 sec o wipe
  - **Menial Laborers** → Crude Weapons DoT stacking
- **Evitare**: 
  - **Cave-In rocks** (charge Wick)
  - **Reckless Charge path** (Wick)
  - Stare troppo lontano se sei furthest player (trigger Wick charge)

## Posizionamento e movimento

- **Tank**: 
  - Posiziona Ol' Waxbeard **al centro arena**
  - Quando Wick spawna → posiziona per far **charge attraverso add stack**
  - Face boss lontano da raid (Rock Buster frontal cone possibile)

- **Target Luring Candleflame**:
  - **Kita add** in posizione centrale
  - Stack add per AoE cleave DPS
  - Porta add verso **percorso Wick charge** per instakill

- **Raid**: 
  - **Spread light** per evitare essere tutti furthest (Wick charge random)
  - Se Wick sta per charge → **move lateralmente** per evitare path
  - Non stare mai in line-of-sight Cave-In (rocce cadenti)

- **Movement minecart (M+)**:
  - Minecart spawna in posizione fissa
  - Focus DPS su minecart immediatamente
  - Evita minecart path se si muove

## Gestione cooldown

- **Lust/Heroism**: pull (boss breve, maximize damage uptime)
- **DPS CD**: su add waves (Menial Laborers) + minecart (M+)
- **Tank CD**: su Rock Buster stack alti (se no tank swap)
- **AoE stun/slow**: su add Menial Laborers per rallentare Crude Weapons stack

## Note per Tank

- **Rock Buster**:
  - Colpo pesante con debuff stacking
  - Se M+ high key → **tank swap** dopo 2-3 stack
  - Usa CD rotazionali su stack alto

- **Positioning critico**:
  - Posiziona Ol' Waxbeard **centro arena**
  - Quando **Wick charge** → posiziona per far charge **attraverso add stack**
  - Cave-In rocks uccidono add → maximize value

- **Add management**:
  - Menial Laborers fixano target Luring Candleflame (non tankabili)
  - Focus su boss uptime, DPS gestisce add

## Note per Healer

- **Rock Buster**:
  - Tank damage spike con debuff stacking
  - Pre-heal tank se stack alti

- **Crude Weapons DoT**:
  - Add Menial Laborers applicano stacking Physical DoT
  - Se add non muoiono veloce → raid damage aumenta esponenzialmente
  - Spot heal giocatori con DoT alto

- **Cave-In rocks**:
  - Se raid colpito da rocks → instant heal spike
  - Rocce istakillano add ma danneggiano giocatori

- **Minecart explosion (M+)**:
  - Se minecart non distrutto entro 20 sec → raid-wide wipe mechanic
  - Prepare raid CD se DPS struggling

## Note per DPS

- **Priority target**:
  1. **Minecart dinamica** (M+) → max 20 sec o wipe
  2. **Menial Laborers** → stack Crude Weapons DoT pericoloso
  3. Ol' Waxbeard (boss damage filler)

- **Luring Candleflame**:
  - Se sei target → **kita add in stack** al centro
  - Non correre random → porta add dove tank/DPS vogliono
  - Stack add per AoE cleave massimo

- **Wick Reckless Charge**:
  - Wick charge verso **furthest player**
  - Se sei furthest → **move lateralmente** per evitare charge
  - **Cave-In rocks lungo path** → evita percorso
  - Usa charge per **instakill add** (posiziona add in path)

- **AoE priority**:
  - Menial Laborers DEVONO morire veloce
  - Crude Weapons DoT stacking = wipe se ignorato
  - Usa AoE stun/slow per rallentare add

## Errori comuni

1. **Non killare minecart (M+)**
   - Minecart ha 20 sec timer
   - Se non distrutto → esplosione raid-wide wipe
   - PRIORITY #1 quando spawna

2. **Ignorare Menial Laborers**
   - Crude Weapons DoT stacking
   - Se add vivono troppo → DoT damage esponenziale = wipe
   - Focus AoE immediato

3. **Target Luring Candleflame kita add male**
   - Add fixano target marcato
   - Se target corre random → add spreadati = no AoE cleave
   - Porta add in **stack centrale** per cleave efficace

4. **Non usare Cave-In per killare add**
   - Wick Reckless Charge causa Cave-In rocks
   - Rocks istakillano add se colpiti
   - Posiziona add in **path charge** per free instakill

5. **Raid sta troppo lontano (trigger charge random)**
   - Wick charge verso furthest player
   - Se raid spread troppo → charge unpredictable
   - Stay **medium range** dal boss per controllare charge target
