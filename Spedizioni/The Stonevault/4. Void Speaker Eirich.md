# Void Speaker Eirich — Boss 4

High Speaker corrotto dal Void. Meccanica chiave: Void Corruption (raid damage se Void Fragments non raccolti), Entropic Reckoning (debuff spread), Void Tendrils add.

## Panoramica boss

**Tipo**: Void Fragment Management + Add Control Boss  
**Meccanica chiave**: Void Corruption (stacking raid damage), Void Fragments (raccogliere riduce damage), Entropic Reckoning (debuff spread), Void Tendrils add priority kill  
**Difficoltà principale**: raccogliere Void Fragments velocemente, spread Entropic Reckoning, kill Void Tendrils add, evitare Void Rifts

## Meccaniche chiave

1. **Void Corruption** (Corruzione del Vuoto)
   - **Debuff passivo** su TUTTO il raid
   - Infligge **stacking Shadow damage** periodico
   - Damage aumenta progressivamente durante fight
   - **Void Fragments** raccogliere riduce stack/damage
   - Se troppi stack → raid overwhelmed

2. **Void Fragments** (Frammenti del Vuoto)
   - Spawn in arena come **orbs raccoglibili**
   - **Raccogli Void Fragment** → riduce Void Corruption stack raid
   - **CRUCIALE**: priority raccogliere fragment velocemente
   - Più fragment raccolti = meno raid damage
   - Fragment spawn periodicamente

3. **Entropic Reckoning** (Resa dei Conti Entropica)
   - Eirich marca **giocatori casuali** con debuff
   - Dopo timer → **esplosione Shadow damage** in 8 yard
   - **Spread lontano da raid** quando marcato
   - Explosion lascia **Void Rift** a terra (zona damage)
   - Se stack → multi-kill

4. **Void Rift** (Fenditura del Vuoto)
   - Pool a terra da Entropic Reckoning esplosione
   - **Persistente** durante fight
   - **Shadow damage DoT** se dentro
   - Riduce spazio safe progressivamente
   - **EVITA Void Rift pool** sempre

5. **Void Tendrils** (Tentacoli del Vuoto)
   - Eirich spawna **add Void Tendrils**
   - Add cast **Void Bolt** verso raid (Shadow damage)
   - **PRIORITY KILL** add immediatamente
   - Interrupt Void Bolt se possibile
   - Se troppi add alive → raid damage overwhelming

6. **Unbridled Void** (Vuoto Sfrenato)
   - Cast raid-wide **Shadow damage** blast
   - **Interrompibile** (priority interrupt)
   - Se cast completo → alto raid damage + Void Corruption stack boost
   - Coordinate interrupt rotation

7. **Void Empowerment** (Potenziamento del Vuoto)
   - Periodicamente Eirich **empowered** da Void
   - Aumenta damage output boss
   - Damage buff stackabile
   - **Burn boss** prima troppi stack

8. **Collapsing Void** (Vuoto Collassante)
   - Cast che pull giocatori **verso boss**
   - Dopo pull → **AoE explosion** in melee range
   - **Melee escape** dopo pull
   - Ranged già safe (fuori melee range)

## Cosa interrompere / dispellare / evitare

- **Interrupt priority**: 
  1. **Unbridled Void** (raid-wide Shadow damage)
  2. **Void Bolt** (Void Tendrils add cast)
- **Killa priority**: 
  - **Void Tendrils add** → priority #1
- **Evitare**: 
  - **Void Rift pools** (da Entropic Reckoning esplosione)
  - **Entropic Reckoning 8 yard radius** (spread)
  - **Collapsing Void AoE** (melee escape dopo pull)
- **Raccogliere**: 
  - **Void Fragments** → riduce Void Corruption stack raid

## Posizionamento e movimento

- **Tank**: 
  - Posiziona Eirich **lontano da raid** (Entropic Reckoning explosion safe)
  - Face boss lontano da raid (possibile frontal)
  - **Raccoglie Void Fragments** se vicino (tank mobile)
  - Durante Collapsing Void → stay in melee (dopo pull escape)

- **Raid**: 
  - **Medium spread** per Entropic Reckoning
  - **Raccogli Void Fragments** appena spawn (priority)
  - Quando **marcato** Entropic Reckoning → **spread lontano 8+ yard**
  - **Evita Void Rift pools** sempre

- **Entropic Reckoning target**:
  - Timer countdown dopo mark
  - **Run lontano da raid** (8+ yard)
  - Esplosione crea Void Rift → posiziona pool in zona non-critical

- **Void Fragments**:
  - Spawn in posizioni random arena
  - **Closest player raccoglie** immediatamente
  - Non tutti devono correre (spread responsibility)

- **Collapsing Void**:
  - Eirich pull tutti verso sé
  - **Melee**: dopo pull → **escape out** veloce (AoE explosion melee range)
  - **Ranged**: già safe (fuori melee)

## Gestione cooldown

- **Raid CD**: 
  - Su Void Corruption alto stack (se fragment collection slow)
  - Durante Unbridled Void se interrupt fail
- **Tank CD**: 
  - Durante Void Empowerment stack alto (increased boss damage)
  - Collapsing Void AoE se troppo lento escape
- **Lust/Heroism**: pull o ~30% HP (burn prima Void Corruption overwhelm)
- **DPS CD**: 
  - Pull (maximize burn)
  - Su Void Tendrils add waves (kill fast)

## Note per Tank

- **Void Fragments**:
  - Tank mobile → raccoglie fragment se vicino
  - **Priority raccolta** riduce raid damage
  - Don't ignore fragment (non solo DPS job)

- **Collapsing Void**:
  - Pull verso boss → dopo **escape melee range** veloce
  - AoE explosion in melee = alto damage
  - Tank cooldown se troppo lento escape

- **Void Empowerment**:
  - Eirich damage aumenta progressivamente
  - Stack alto → tank buster risk
  - Rotate defensive CD durante stack alto

- **Positioning**:
  - Boss lontano da raid (Entropic Reckoning safe spread)
  - Face boss lontano da raid
  - Non tank boss vicino a Void Rift pools

## Note per Healer

- **Void Corruption**:
  - **Stacking raid damage** passivo
  - Healing pressure aumenta durante fight
  - Se Void Fragments non raccolti → overwhelming healing
  - **Raccogli fragment** se vicino (help raid)

- **Entropic Reckoning**:
  - Target marcato → esplosione 8 yard
  - Pre-shield target prima esplosione
  - Spot heal dopo explosion

- **Void Tendrils**:
  - Add spam Void Bolt → raid damage stack
  - Se troppi add → healing impossible
  - Call out kill add priority

- **Unbridled Void**:
  - Se interrupt fail → raid-wide damage spike
  - Pre-shield raid se unsure interrupt

## Note per DPS

- **Void Fragments**:
  - **PRIORITY raccogliere** appena spawn
  - Più fragment raccolti = meno raid damage
  - Closest player responsabilità
  - Non ignorare fragment (critico survival)

- **Void Tendrils**:
  - **PRIORITY KILL** add immediatamente
  - Add spam Void Bolt = raid damage
  - AoE/cleave add veloce
  - Interrupt Void Bolt se possibile

- **Entropic Reckoning**:
  - Se marcato → **spread 8+ yard** lontano raid
  - Posiziona explosion in zona safe (no critical path)
  - Void Rift pool persiste → minimize bad pool placement

- **Interrupt rotation**:
  - Coordinate Unbridled Void interrupt
  - Multiple interrupters (rotation)
  - Priority interrupt se disponibile

- **Burn priority**:
  - Lust pull, maximize DPS
  - Void Corruption stack = timer soft enrage
  - Finish fight prima Void Corruption overwhelm healer

## Strategia chiave

1. **Phase flow**:
   - Pull → raid spread medium
   - Void Fragments → raccogliere priority
   - Entropic Reckoning → spread target lontano 8+ yard
   - Void Tendrils → kill add immediatamente
   - Unbridled Void → interrupt priority
   - Collapsing Void → melee escape dopo pull
   - Void Corruption → burn boss prima stack overwhelm

2. **Priority list**:
   - Raccogliere Void Fragments (riduce Void Corruption)
   - Kill Void Tendrils add (priority #1)
   - Interrupt Unbridled Void
   - Spread Entropic Reckoning target
   - Evitare Void Rift pools
   - Burn boss prima Void Corruption overwhelm

3. **Common mistakes**:
   - Ignorare Void Fragments → Void Corruption overwhelming raid
   - No kill Void Tendrils → add damage overwhelming
   - Stack durante Entropic Reckoning → multi-kill
   - Miss Unbridled Void interrupt → raid wipe risk
   - Melee no escape Collapsing Void → AoE death
   - Troppi Void Rift pool mal posizionati → no spazio safe
